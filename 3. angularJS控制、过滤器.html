控制器，controller

控制器是一个JavaScript对象？在我看来，更像是定义好的函数，我们传入需要控制的那部分HTML区域，接着定义一个feedback来操作，添加或修改属性。通用代码如下：

            app.controller('myCtrl', function(){
                //  do something
            });

myCtrl定义在html标记的ng-controller里，以控制这块代码，后面那个匿名函数来添加或修改属性

  • {{ name }}
  • {{ phone }}
  • {{ getInfomation() }}
